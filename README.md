Spotify Web Player Clone

A responsive Spotify-inspired music player interface built using HTML5 and CSS3. The project recreates the look and feel of Spotify's desktop web player, focusing on responsive design, reusable UI components, and modern CSS styling.

🚀 Features
Responsive Spotify-inspired user interface
Sidebar navigation
Sticky navigation bar
Music player layout
Album information section
Playlist cards
Hover animations
Custom progress bar
Font Awesome icons
Google Fonts integration
